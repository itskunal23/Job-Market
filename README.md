# RoleWithAI: Your Career Co-Pilot

A supportive PWA that helps job seekers navigate the 2026 job market with truth scores, ghost risk indicators, and a human-first approach to job searching.

## Features

- **Morning Brief**: Daily market insights with hiring trends and ghosting rates
- **Truth Scores**: 1-100 score for each job posting based on real hiring intent
- **Ghost Risk Meter**: Low/Medium/High indicators to help you prioritize applications
- **Status Ledger**: Timeline of your applications with RoleWithAI's personalized notes
- **Talk to RoleWithAI**: Interactive chat to get insights on why you should (or shouldn't) apply

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Design System

- **Colors**: Warm Sage (#649b64), Pearl White (#f5f5f5), Deep Charcoal (#1a1a1a)
- **Style**: Airy minimalist with glass morphism cards
- **Typography**: Serif headers (Georgia), Sans-serif body text
